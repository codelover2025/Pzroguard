CHAPTER 3: CODING OF FUNCTIONS - FULL 18 FEATURES

This chapter details **all 18 major modules** of ProGuard, the AI-Based Fake Productivity Detection System. Each 3.x subsection covers implementation, algorithms, and sample code from source files. Python + OpenCV, DeepFace, psutil, matplotlib, reportlab, etc.

3.1 Mouse & Keyboard Pattern Detection (src/proguard/collectors/mouse_keyboard.py)
Monitors input to detect macros/jigglers via entropy.
Algorithms: Shannon entropy (mouse angles/key intervals), macro via variance<0.1.
Sample Code:
```python
collector = MouseKeyboardCollector()
# on_mouse_move, on_key_press callbacks
metrics = collector.get_current_metrics()
# {'mouse_entropy': 0.68, 'keyboard_entropy': 0.72, 'macro_detected': False}
```

3.2 Screen Activity Monitor (src/proguard/collectors/screen_activity.py)
Tracks active apps/windows via psutil/win32gui, productive_ratio.
Algorithms: PRODUCTIVE_APPS classification, time tracking.
Sample Code:
```python
monitor = ScreenActivityMonitor()
metrics = monitor.get_current_metrics()
# {'productivity_ratio': 0.76, 'current_app': 'code.exe', 'top_apps': [...]}
```

3.3 Typing Rhythm Analysis (src/proguard/analytics/typing_rhythm.py)
LSTM/entropy for human vs bot typing.
Algorithms: Intervals normalized, LSTM sigmoid 0-1, rhythm_entropy.
Sample Code:
```python
analyzer = TypingRhythmAnalyzer()
analyzer.add_keystroke(time.time(), 'a')
metrics = analyzer.get_current_metrics()
# {'typing_authenticity': 0.92, 'rhythm_entropy': 0.85}
```
